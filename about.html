About TrendBoard

TrendBoard is a free, static website that helps creators find trends, build better content, and discover local & global topics. Built to run on GitHub Pages with minimal maintenance.

Contact & Contribute

Want to add a tool or suggest data sources? Open an issue on the repo. Community contributions use GitHub Issues.

Note: TrendBoard uses public endpoints and runs client-side. Do not submit private data via issues.
